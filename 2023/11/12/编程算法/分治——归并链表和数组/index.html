分治——归并链表和数组

C++ 链表 数组 递归
编程算法
发布日期:   2023-11-12
更新日期:   2023-11-12

这篇博客会介绍如何通过分治的方法来归并排序链表和数组，其实归并的基本思想就是分治，通过不断地将大的问题变成小问题，就是将大的链表合并从两个元素的合并开始一步步向上归并，是一种常见的后序递归

基本思路如下：

①创建一个递归函数，我把它叫做dispatch，它的作用就是向下递归把大的合并变成两个元素的合并，一般它的return语句都是调用归并函数，参数一般就是传入原数据结构和left和right两个参数

②创建一个合并两个有序结构的函数，注意必须是有序的！其返回值是合并后的结构

③主函数中只需要调用一次dispatch函数即可

23. 合并 K 个升序链表 - 力扣（LeetCode）

class Solution {
public:
    ListNode* mergeTwoLists(ListNode *a, ListNode *b) { //合并两个有序链表
        if ((!a) || (!b)) return a ? a : b;
        ListNode head, *tail = &head, *aPtr = a, *bPtr = b;
        while (aPtr && bPtr) {
            if (aPtr->val < bPtr->val) {
                tail->next = aPtr; aPtr = aPtr->next;
            } else {
                tail->next = bPtr; bPtr = bPtr->next;
            }
            tail = tail->next;
        }
        tail->next = (aPtr ? aPtr : bPtr);
        return head.next;//返回合并后的链表
    }

    ListNode* dispatch(vector <ListNode*> &lists, int l, int r) {//分而治之
        if (l == r) return lists[l];//递归结束条件就是返回这个元素本身
        if (l > r) return nullptr;
        int mid = (l + r) >> 1;
        //最后的return就是把原本一整个链表的归并分解成左右两个链表
        return mergeTwoLists(dispatch(lists, l, mid), dispatch(lists, mid + 1, r));
    }

    ListNode* mergeKLists(vector<ListNode*>& lists) {//主函数，只需要调用一次dispatch
        return merge(lists, 0, lists.size() - 1);
    }
};

数组的==原地==归并排序

原地排序和上面合并K个链表的不同在于只在原地修改数组而不创建新数组

void merge2vec(vector<int>& vec, vector<int>& tmp1, vector<int>& tmp2){//原地归并两个数组
    int len1 = tmp1.size();
    int len2 = tmp2.size();
    int p1 = 0;
    int p2 = 0;
    while(p1 < len1 && p2 < len2){
        if(tmp1[p1] <tmp2[p2]){
            vec.push_back(tmp1[p1++]);
        }else{
            vec.push_back(tmp2[p2++]);
        }
    }
    while(p1 <len1){
        vec.push_back(tmp1[p1++]);
    }
    while(p2 < len2){
        vec.push_back(tmp2[p2++]);
    }
}
void dispatch(vector<int>& vec){//dispatch，思路和链表的一样，只不过变成了数组
    if(vec.size() <= 1){
        return;
    }
    auto mid = vec.begin() + vec.size()/2;
    vector<int> tmp1(vec.begin(), mid);
    vector<int> tmp2(mid, vec.end());
   
    vec.clear();
    merge2vec(vec, dispatch(tmp1), dispatch(tmp2));//理解这段就明白了，最重要的递归
}

通过这两个分治法可以看到，都是

merge2(dispatch,dispatch)

的形式

也就是mergesort会返回结构，而dispatch是往下分治的！dispatch不需要写复杂的代码！

==merge函数里只需要写下一次分治的mid和递归结束的条件就好了==

对单个链表用归并排序

和前面不同的是，需要通过快慢指针找到中点

148. 排序链表 - 力扣（LeetCode）

class Solution {
public:
    ListNode* mergeList(ListNode* one, ListNode* two) {//归并两个链表
        ListNode node, *tail = &node;
        while (one && two) {
            if (one->val < two->val) {
                tail->next = one;
                one = one->next;
            } else {
                tail->next = two;
                two = two->next;
            }
            tail = tail->next;
        }
        tail->next = one ? one : two;
        return node.next;
    }
    void splitList(ListNode* head, ListNode*& one, ListNode*& two) { //找到中位点并把中点和起点赋予one和two
        ListNode *s = head, *f = head;
        while (f->next) {//快慢指针找到中位链表，将链表左右拆分变成两条链表
            f = f->next;
            if (f->next) {
                f = f->next;
                s = s->next;
            }
        }
        one = head;
        two = s->next;
        s->next = nullptr;
    }

    ListNode* dispatch(ListNode* head) {//dispatch
        if (!head || !head->next) return head;//dispatch最小单位
        ListNode *one = nullptr, *two = nullptr;
        splitList(head, one, two);//链表拆分两半
        return mergeList(sortList(one), sortList(two));
    }
};
