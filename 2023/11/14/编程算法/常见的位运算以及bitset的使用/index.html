常见的位运算以及bitset的使用

C++ 位运算
编程算法
发布日期:   2023-11-14
更新日期:   2023-11-17
文章字数:   1.2k
阅读时长:   5 分
阅读次数:  

C++中有许多位运算操作符和位运算函数，它们用于在二进制级别上操作整数类型的数据。

操作符

  1. 按位与（&）： 对应位上的两个操作数都为1时，结果为1，否则为0。

    int result = a & b;

  2. 按位或（|）： 对应位上的两个操作数只要有一个为1，结果就为1。

    int result = a | b;

  3. 按位异或（^）： 对应位上的两个操作数相异时，结果为1，否则为0。

    int result = a ^ b;

  4. 按位取反（~）： 对操作数的每一位取反。

    int result = ~a;

  5. 左移（<<）： 将操作数的所有位向左移动指定的位数，右侧用0填充。

    int result = a << n;

  6. 右移（>>）： 将操作数的所有位向右移动指定的位数，左侧用符号位（对于有符号数）或者0填充。

    int result = a >> n;

常用示例：

①获得a的末尾

int temp = a & 1;

②让a左移一位

a<<=1;

bitset的使用

在C++中，你可以使用std::bitset类来创建一个包含固定数量的二进制位的bitset。你可以使用以下几种方法来初始化std::bitset：

  1. 使用默认构造函数初始化：
std::bitset<8> bits; // 创建一个包含8个位的bitset，所有位都被初始化为0
  1. 使用整数值初始化：
std::bitset<8> bits(12); // 使用整数12初始化bitset，结果为00001100
  1. 使用二进制字符串初始化：
std::bitset<8> bits("10101010"); // 使用二进制字符串初始化bitset
  1. 使用十六进制字符串初始化：
std::bitset<8> bits(0xAA); // 使用十六进制值初始化bitset，结果为10101010
  1. 使用另一个bitset初始化：
std::bitset<8> originalBits("10101010");
std::bitset<8> bits(originalBits); // 使用另一个bitset进行初始化
  1. 使用部分位初始化：
std::bitset<8> bits;
bits[0] = 1;
bits[2] = 1;
bits[4] = 1;
// 结果为10101000

无论使用哪种初始化方法，你都可以通过位运算和其他操作来处理和修改std::bitset中的位。这些位可以通过下标操作符[]来访问，也可以使用成员函数来执行各种操作，如set()、reset()、flip()等。

以下是std::bitset类的一些常用成员函数和操作：

  1. std::bitset<N>::bitset()：默认构造函数，创建一个包含N个位的bitset，所有位都被初始化为0。

  2. std::bitset<N>::bitset(unsigned long val)：使用整数值初始化bitset。

  3. std::bitset<N>::bitset(const std::string& str)：使用二进制字符串初始化bitset。

  4. ==std::bitset<N>::to_string()：将bitset转换为二进制字符串。==

  5. std::bitset<N>::operator[]：通过下标操作符访问bitset中的位。

  6. std::bitset<N>::size()：返回bitset的位数。

  7. std::bitset<N>::count()：返回bitset中设置为1的位的数量。

  8. std::bitset<N>::test(size_t pos)：测试指定位置的位是否为1。

  9. std::bitset<N>::any()：检查bitset中是否有至少一个位被设置为1。

  10. std::bitset<N>::none()：检查bitset中是否没有任何位被设置为1。

  11. std::bitset<N>::all()：检查bitset中是否所有位都被设置为1。

  12. std::bitset<N>::set()：将所有位设置为1。

  13. std::bitset<N>::set(size_t pos)：将指定位置的位设置为1。

  14. std::bitset<N>::reset()：将所有位重置为0。

  15. std::bitset<N>::reset(size_t pos)：将指定位置的位重置为0。

  16. ==std::bitset<N>::flip()：将所有位取反。==

  17. std::bitset<N>::flip(size_t pos)：将指定位置的位取反。

  18. ==std::bitset<N>::to_ulong()：将bitset转换为unsigned long类型。==

  19. ==std::bitset<N>::to_ullong()：将bitset转换为unsigned long long类型。==

  20. std::bitset<N>::operator<<：左移操作，将bitset左移指定位数。

  21. std::bitset<N>::operator>>：右移操作，将bitset右移指定位数。

这些成员函数允许你在bitset上执行各种位操作，包括设置、重置、翻转、测试和检查位状态等。你可以根据需要使用这些函数来操作bitset中的位。

最好不要对bitset成员使用auto创建！

bitset<1> a(0);
auto temp1 = a[0];//temp1 = 0
int temp2 = a[0];//temp2 = 0;
a[0] = 1;//此时temp1也会变成1！！！而temp2不变！！

补码

  • 补码：如果是正数，则表示方法和原码一样；如果是负数，则将其反码加上 1（相当于将原码数值位取反,然后在最低位加 1）。
  • 正数的原码、反码、补码完全一样，只有负数需要按照以上规则计算。
  • 0 的反码、补码都为 0。

如计算 -3 的二进制补码，先计算 3 的原码 0000 0011，取反得到 1111 1100，再加 1 为 1111 1101。

在左移负数的时候，不能以a==0作为判断条件，因为负数无论怎么左移，都是补的1
